#  Whatsapp Chat Analyzer

🚀 Introduction

Ever wondered how much you chat on WhatsApp? Who sends the most messages? Which emoji do you use the most? WhatsApp Chat Analyzer is a powerful tool that extracts insights from your chat history!

<br/>

**✨ Features**

✅ User Activity Analysis – Find the most active users in a group chat 📈.

✅ Word Frequency Analysis – Discover the most commonly used words 🔤.

✅ Emoji Statistics – See which emojis are used the most 🤩🔥.

✅ Chat Timeline – Visualize message trends over time 📅.

✅ Most Active Days & Hours – Identify peak chatting times ⏰.

<br/>

🛠️ **Technologies Used**

. Python 🐍 – Core programming language

. Streamlit – Web UI for interactive analysis

. Pandas – Data manipulation

. Matplotlib & Seaborn – Data visualization 📊

. Regular Expressions (Regex) – Message parsing

<br/>

🔍 **Data Science**

The app processes chat data using Pandas to clean and structure the raw chat logs into a tabular format.
Data wrangling techniques (e.g., tokenization, stopword removal) are applied to make analysis meaningful.

📈 **Data Analytics**

Computes statistical summaries like the number of messages per user, word frequency, and emoji usage.
Generates visual insights using Matplotlib & Seaborn, including pie charts and bar graphs.

<br/>

**📝 How to Use**

1. Export your chat from WhatsApp as a .txt file (without media).
2. Upload the file in the web app.
3. Explore the insights using interactive graphs and visualizations!
